## Overview

SIM LPPM ITSNU is designed to align with **BIMA (Basis Data Ilmu Pengetahuan Indonesia)** - the national research information system from Kemdikbudristek. This document provides the official master data classifications used in BIMA for implementation in the local system.

## 4. Implementation in SIM LPPM ITSNU

### Database Seeding Strategy

**1. Research Schemes**
```sql
-- Seed research_schemes table with BIMA codes
INSERT INTO research_schemes (name, strata, description) VALUES
('Penelitian Dosen Pemula (PDP)', 'Dasar', 'Early Career Faculty Research'),
('Penelitian Fundamental (PF)', 'Dasar', 'Fundamental Research'),
('Penelitian Terapan (PT)', 'Terapan', 'Applied Research'),
('Pemberdayaan Wilayah (PW)', 'PKM', 'Regional Empowerment'),
-- Add all schemes from BIMA...
```

**2. National Priorities**
```sql
-- Seed national_priorities with 9 PRN focus areas
INSERT INTO national_priorities (name, description) VALUES
('Pangan', 'Riset ketahanan pangan dan pertanian'),
('Energi', 'Riset energi baru dan terbarukan'),
('Kesehatan', 'Riset kesehatan dan obat'),
-- Add all 9 PRN areas...
```

**3. Science Clusters (3-Level)**
```sql
-- Level 1
INSERT INTO science_clusters (name, level, parent_id) VALUES
('Matematika dan Ilmu Pengetahuan Alam (MIPA)', 1, NULL),
('Ilmu Teknik', 1, NULL),
-- Add all 12 Level 1...

-- Level 2 (under MIPA)
INSERT INTO science_clusters (name, level, parent_id) VALUES
('Ilmu IPA', 2, (SELECT id FROM science_clusters WHERE name = 'Matematika dan Ilmu Pengetahuan Alam (MIPA)')),
-- Add all Level 2...

-- Level 3 (under Ilmu IPA)
INSERT INTO science_clusters (name, level, parent_id) VALUES
('Fisika', 3, (SELECT id FROM science_clusters WHERE name = 'Ilmu IPA')),
('Kimia', 3, (SELECT id FROM science_clusters WHERE name = 'Ilmu IPA')),
-- Add all Level 3...
```

## 5. Alignment Benefits

### For ITSNU Internal
✅ Standardized taxonomy for institutional reporting  
✅ Consistent classification across proposals  
✅ Easier data aggregation and analysis  
✅ Faculty can find relevant categories easily  

### For National Integration
✅ Compatible with BIMA reporting requirements  
✅ Aligned with national research priorities (PRN)  
✅ Follows OECD FoS international standards  
✅ Ready for future BIMA data synchronization  

### For Accreditation
✅ Meets BAN-PT classification requirements  
✅ Supports institutional research profiling  
✅ Facilitates research output reporting  
✅ Aligns with national research roadmap
