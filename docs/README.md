# Dokumentasi SIM LPPM ITSNU

Selamat datang di pusat dokumentasi SIM LPPM ITSNU Pekalongan. Dokumentasi ini disusun secara teratur untuk membantu pemangku kepentingan memahami visi, alur kerja, dan aspek teknis dari sistem ini.

## Daftar Dokumen Terurut

1.  **[01. Product Requirements Document (PRD)](01-prd.md)**
    Visi, masalah yang diselesaikan, tujuan strategis, dan fitur utama aplikasi.
    
2.  **[02. Workflow Lengkap](02-workflow-lengkap.md)**
    Panduan alur bisnis mulai dari pengajuan proposal hingga penetapan status akhir.
    
3.  **[03. Peran dan Wewenang](03-peran-dan-wewenang.md)**
    Matriks tanggung jawab (RACI) dan hak akses data untuk setiap peran pengguna.
    
4.  **[04. Struktur Data](04-struktur-data.md)**
    Perbedaan teknis dan substantif antara kegiatan Penelitian dan Pengabdian (PKM).
    
5.  **[05. Transisi Status Proposal](05-transisi-status.md)**
    Penjelasan teknis mengenai mesin status (*state machine*) dan validasi perpindahan status.
    
6.  **[06. Data Master & Referensi](06-master-data.md)**
    Katalog taksonomi, bidang ilmu, skema, dan referensi anggaran yang digunakan sistem.
    
7.  **[07. Sistem Notifikasi](07-notifikasi.md)**
    Daftar pemicu notifikasi otomatis baik melalui aplikasi maupun email.
    
8.  **[08. Entity Relationship Diagram (ERD)](08-erd.md)**
    Gambaran teknis struktur basis data dan relasi antar entitas utama.

---
*Gunakan dokumen ini sebagai panduan resmi dalam pengembangan dan operasional SIM LPPM ITSNU.*
