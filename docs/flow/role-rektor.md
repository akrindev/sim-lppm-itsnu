# Alur Kerja Peran: Rektor (University Rector)

Rektor memiliki akses pengawasan strategis terhadap seluruh kegiatan penelitian dan pengabdian masyarakat di universitas.

## 1. Pengawasan Strategis
*   **Dashboard Statistik**: Melihat ringkasan data penelitian dan PKM secara global (jumlah usulan, status, sebaran fakultas).
*   **Laporan**: Mengakses laporan perkembangan kegiatan LPPM.

## 2. Pengambilan Kebijakan
*   Menggunakan data dari SIM LPPM untuk menentukan arah kebijakan riset dan pengabdian masyarakat di tingkat universitas.

---
**Status Terkait:** Melihat semua status.
