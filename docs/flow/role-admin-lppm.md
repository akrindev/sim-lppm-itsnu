# Alur Kerja Peran: Admin LPPM (LPPM Staff)

Admin LPPM bertanggung jawab atas operasional teknis proses review dan pengelolaan data pendukung.

## 1. Penugasan Reviewer
*   **Monitoring Antrian**: Memantau usulan dengan status `WAITING_REVIEWER`.
*   **Pilih Reviewer**: Memilih dosen/pakar yang memiliki peran `reviewer` untuk menilai usulan tersebut.
*   **Set Deadline**: Menentukan batas waktu pengerjaan review (default: 14 hari).
*   **Aktifkan Review (`UNDER_REVIEW`)**: Setelah reviewer ditugaskan, status usulan otomatis berubah menjadi `UNDER_REVIEW`.

## 2. Pengelolaan Data Master
*   **Manajemen Skema**: Mengatur jenis-jenis penelitian dan PKM.
*   **Data Pendukung**: Mengelola data Fakultas, Program Studi, Fokus Bidang, dan SBK (Standar Biaya Keluaran).
*   **Pengaturan TKT**: Mengelola indikator Tingkat Kesiapan Teknologi.

## 3. Monitoring Progres
*   Memantau jalannya proses review dan memastikan reviewer menyelesaikan tugas tepat waktu.

---
**Status Terkait:** `WAITING_REVIEWER`, `UNDER_REVIEW`.
